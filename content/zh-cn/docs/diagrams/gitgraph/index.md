需求图以可视化的方式展现需求及其相互联系，以及与其他文档化元素之间的联系。建模规范遵循 SysML v1.6 的定义。


Git 图是 Git 提交和 Git 操作（命令）在各个分支上的图形化表示。

这类图表对于开发人员和运维团队分享 Git 分支策略尤其有帮助。例如，它可以更直观地展示 Git 的工作流程。

Mermaid 可以渲染 Git 图。
